== Description ==

The plugin allows the WordPress site administrator to easily control access to pages based on the user's role.

Select the page / taxonomy types in the plugin settings and after that you can restrict access to users according to their roles.


== Features ==

*  Support for standard and custom pages and posts
*  Support for standard and custom taxonomies
*  Custom Redirect
*  Setting actions for guests (visitors)

== Installation ==

1. Upload the 'access-pages-by-role-for-admin' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on 'Setting' - 'Access Pages Control' in the left admin bar of your dashboard
